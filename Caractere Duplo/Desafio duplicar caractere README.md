# Caractere Duplo

## Descrição do problema

Dada uma string, o objetivo é gerar uma nova string onde **cada caractere apareça duas vezes**, respeitando exatamente letras maiúsculas, minúsculas, números e símbolos.

Ou seja, cada elemento da string original deve ser duplicado mantendo a mesma ordem.

## Exemplos

Entrada:
"String"
Saída:
"SSttrriinngg"

Entrada:
"Hello World"
Saída:
"HHeelllloo  WWoorrlldd"

Entrada:
"1234!_"
Saída:
"11223344!!__"

## O que este desafio trabalha

Este problema exercita a manipulação de strings em Python e o uso de estruturas de repetição para percorrer cada caractere individualmente.

Também reforça a ideia de construção progressiva de uma nova saída a partir de uma entrada existente, algo muito comum no tratamento de dados em programação.
