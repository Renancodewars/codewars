# codewars
# Resolução de Problemas em Python (Codewars)

Este repositório contém soluções que desenvolvi durante meus estudos de lógica de programação e Python através de desafios de programação.

Objetivo:
Praticar raciocínio lógico, manipulação de strings, listas, loops e estruturas condicionais.

Cada pasta representa um problema diferente.
Dentro de cada uma existe:

* o código da solução
* uma explicação simples da lógica utilizada

Tecnologias utilizadas:

* Python 3

O foco deste repositório não é apenas resolver o problema, mas organizar e explicar o raciocínio por trás da solução.
